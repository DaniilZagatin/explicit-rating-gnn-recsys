# Схемы моделей

В этой папке лежат изображения, которые используются для пояснения архитектур моделей в README и документации проекта.  

## Список файлов

| Файл | Описание |
|---|---|
| `SASRec_like.png` | Схема двубашенной модели SASRec-вида. Пользовательское представление строится по последовательности просмотренных/оценённых фильмов с помощью Transformer-блока, а представление кандидата строится отдельной item-башней. Итоговый score считается через скалярное произведение user и item embedding. |
| `GCMC.png` | Схема GCMC-подобной модели. User-item граф кодируется графовым encoder, после чего bilinear decoder восстанавливает оценки или score для пар пользователь–фильм. |
| `message_pass.png` | Иллюстрация message passing. Показано, как представление узла обновляется через соседей на разных hop-уровнях. Эта схема используется для объяснения общей идеи графового распространения. |
| `NGCF.png` | Схема NGCF-подобной модели. Показаны propagation-слои, объединение embedding-векторов разных уровней и prediction layer для оценки пары пользователь–фильм. |
