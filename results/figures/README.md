# Графики результатов

В этой папке лежат графики, полученные по результатам экспериментов.  
Они используются для быстрого сравнения моделей по качеству, устойчивости и времени оценки.

## Список файлов

| Файл | Описание |
|---|---|
| `all_models_warm_hr20.png` | Сравнение всех моделей по `Warm HR@20`. График даёт общий вид качества в warm-start сценарии. |
| `all_models_cold_user_hr20.png` | Сравнение моделей по `Cold-user HR@20`. Используется для анализа качества на новых пользователях. |
| `warm_vs_cold_user_tradeoff.png` | Сравнение `Warm HR@20` и `Cold-user HR@20`. По графику видно, какие модели лучше сохраняют качество при переходе от warm-start к cold-user сценарию. |
| `quality_time_tradeoff.png` | Сравнение качества и времени оценки на 100k пользователей. График нужен для анализа компромисса между качеством и вычислительной стоимостью. |
| `validation_tt_vs_ens_gcmc.png` | Динамика качества на валидации для двубашенной модели и гибридного ансамбля `Ens-TT-GCMC-ft`. |
| `rating_weight_learned_function.png` | Функции веса ребра `w(r)` для NGCF-подобной модели. Показывает, как разные варианты учитывают значение рейтинга при распространении сигнала в графе. |

## Как читать графики

Основные обозначения моделей:

| Обозначение | Смысл |
|---|---|
| `MF` | Матричная факторизация |
| `TT` | Двубашенная модель |
| `TT-SASRec` | Двубашенная модель SASRec-вида |
| `GCMC` | GCMC-подобная графовая модель |
| `NGCF` | NGCF-подобная графовая модель |
| `TT<-GCMC` | Двубашенная модель, инициализированная или дообученная с использованием GCMC-представлений |
| `Ens-TT-GCMC-frz` | Ансамбль двубашенной модели и GCMC с замороженными компонентами |
| `Ens-TT-GCMC-ft` | Ансамбль двубашенной модели и GCMC с совместным дообучением |
| `TT-bin` | Вариант двубашенной модели с бинарной постановкой релевантности |

## Основные метрики

В графиках используются:

- `HR@20` — HitRate@20;
- `Warm HR@20` — качество в warm-start сценарии;
- `Cold-user HR@20` — качество для новых пользователей;
- время оценки на 100k пользователей — приближённая оценка вычислительной стоимости инференса.
