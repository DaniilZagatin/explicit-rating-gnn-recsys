# Таблицы результатов

В этой папке собраны таблицы с результатами экспериментов.  
Основные файлы сохранены в формате `.tex`

Таблицы разделены на несколько групп:

- итоговое сравнение моделей;
- результаты по warm-start и cold-user сценариям;
- сравнение отдельных семейств моделей;
- ablation-эксперименты.

## Главные таблицы

Эти таблицы дают общее представление о результатах работы и нужны для финального сравнения моделей.

| Файл | Назначение |
|---|---|
| `final_results_relative_gain.tex` | Главное итоговое сравнение моделей. В таблице приведены результаты для warm-start и cold-user сценариев, относительный прирост к базовой двубашенной модели и время оценки. |
| `final_quality_time.tex` | Сравнение качества и времени оценки. Таблица нужна для анализа компромисса между качеством рекомендаций и вычислительной стоимостью. |
| `full_warm_cold_results.tex` | Полная таблица результатов для warm-start и cold-user сценариев. Удобна как основная расширенная таблица для приложения или подробного анализа. |
| `cold_user_results.tex` | Компактное сравнение моделей в cold-user сценарии. Показывает, какие модели лучше работают для новых пользователей. |
| `hybrid_ensembles.tex` | Сравнение гибридных моделей, объединяющих двубашенную модель и графовые представления. |
| `hybrid_ensembles_with_gain.tex` | То же сравнение гибридных моделей, но с относительным приростом относительно базовой модели. |

## Полные таблицы

Эти файлы содержат подробные результаты по отдельным сценариям.

| Файл | Назначение |
|---|---|
| `full_warm_results.tex` | Полные результаты в warm-start сценарии. |
| `full_cold_user_results.tex` | Полные результаты в cold-user сценарии. |
| `full_warm_cold_results.tex` | Объединённая таблица для warm-start и cold-user сценариев. |

## Таблицы по семействам моделей


| Файл | Назначение |
|---|---|
| `pure_models.tex` | Сравнение отдельных негибридных моделей: MF, двубашенная модель, GCMC и NGCF-подобные варианты. |
| `graph_models.tex` | Сравнение графовых моделей и их модификаций. |
| `graph_models_with_gain.tex` | Сравнение графовых моделей с относительным приростом. |
| `hybrid_ensembles.tex` | Сравнение гибридных ансамблей. |
| `hybrid_ensembles_with_gain.tex` | Сравнение гибридных ансамблей с относительным приростом. |

## Ablation-таблицы

Эти таблицы показывают влияние отдельных решений в моделях.

| Файл | Назначение |
|---|---|
| `features_mf.tex` | Влияние размерности embedding и использования признаков фильмов в матричной факторизации. |
| `features_mf_with_gain.tex` | То же сравнение для MF с относительным приростом. |
| `features_gcmc.tex` | Влияние признаков фильмов в GCMC-подобной модели. |
| `features_gcmc_with_gain.tex` | Влияние признаков фильмов в GCMC с относительным приростом. |
| `rating_weight_function.tex` | Сравнение функций веса ребра в NGCF-подобной модели. |
| `rating_weight_function_with_gain.tex` | Сравнение функций веса ребра с относительным приростом. |
| `temporal_decay.tex` | Влияние временного затухания в GCMC-подобной модели. |
| `temporal_decay_with_gain.tex` | Влияние временного затухания с относительным приростом. |

## Обозначения моделей

| Обозначение | Расшифровка |
|---|---|
| `MF` | Матричная факторизация |
| `TT` | Двубашенная модель |
| `TT-SASRec` | Двубашенная модель SASRec-вида |
| `TT-bin` | Двубашенная модель с бинарной постановкой релевантности |
| `GCMC` | GCMC-подобная графовая модель |
| `GCMC-time` | GCMC-подобная модель с временным затуханием |
| `NGCF-lin` | NGCF-подобная модель с линейной функцией веса ребра |
| `NGCF-exp` | NGCF-подобная модель с экспоненциальной функцией веса ребра |
| `NGCF-spline` | NGCF-подобная модель с обучаемой softplus-spline функцией веса |
| `TT<-GCMC` | Двубашенная модель, дообученная с использованием GCMC-представлений |
| `TT<-NGCF` | Двубашенная модель, дообученная с использованием NGCF-представлений |
| `Ens-TT-GCMC-frz` | Ансамбль двубашенной модели и GCMC с замороженными компонентами |
| `Ens-TT-GCMC-ft` | Ансамбль двубашенной модели и GCMC с совместным дообучением |
| `Ens-TT-NGCF` | Ансамбль двубашенной модели и NGCF-подобной модели |

## Метрики

| Метрика | Описание |
|---|---|
| `HR@20`, `HR@50` | Доля пользователей, для которых хотя бы один релевантный фильм попал в top-K |
| `Recall@20`, `Recall@50` | Доля найденных релевантных фильмов среди всех релевантных фильмов пользователя |
| `nDCG@20`, `nDCG@50` | Качество ранжирования с учётом позиции релевантных фильмов |
| `Время на 100k, сек` | Время оценки модели на 100 тысяч пользователей |
