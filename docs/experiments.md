# Эксперименты

Эксперименты организованы как последовательное сравнение моделей: от простых baseline до графовых и гибридных моделей.

## Основная линия

1. Подготовка датасета и сплит по времени.
2. EDA подготовленных таблиц.
3. Обучение baseline-моделей.
4. Обучение графовых моделей с учётом рейтинга.
5. Обучение гибридных ансамблей.
6. Сравнение качества по warm-start и cold-start сценариям.

## Метрики

Используются стандартные top-K метрики:

```text
Precision@K
Recall@K
HitRate@K
MRR@K
MAP@K
NDCG@K
```

## Сценарии оценки

- `warm-start`: пользователи и фильмы присутствуют в обучающем графе;
- `cold-user`: новый пользователь представлен через support-историю;
- `cold-item`: новый фильм представлен через признаки объекта;
- `both-cold`: одновременно новый пользователь и новый фильм.

## Схемы учёта рейтинга

В разных экспериментах рейтинг используется несколькими способами:

- как бинарная релевантность при `rating >= 4`;
- как graded relevance для NDCG;
- как тип ребра в GCMC;
- как вес ребра в NGCF-подобной модели;
- как дополнительный сигнал при ансамблировании.
