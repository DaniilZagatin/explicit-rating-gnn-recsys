# Систематизация ноутбуков

В репозитории оставлена одна основная линия ноутбуков. Она отражает порядок работы над проектом: от подготовки данных до финального сравнения моделей.

| Файл | Роль |
|---|---|
| `00_dataset_preparation.ipynb` | Подготовка датасета, генерация дополнительных признаков, сплит по времени |
| `01_dataset_eda.ipynb` | Анализ подготовленного датасета |
| `02_baselines_mf_two_tower_gcmc.ipynb` | Базовые модели: матричная факторизация, двубашенная сеть SASRec-вида, GCMC |
| `03_ngcf_rating_weights.ipynb` | NGCF-подобная модель с учётом рейтинга как веса ребра |
| `04_hybrid_two_tower_gcmc.ipynb` | Гибридные модели: ансамблирование двубашенной модели и GCMC |
